# `restobook`

A simple project to manage reservations from restaurants.


## Routes

```
GET   /restaurants/
POST  /restaurants/
GET   /restaurants/23
PUT   /restaurants/23
POST  /reservations
GET   /reservations/2

/search ==> JSON of restaurants to map
/       ==> book restaurant
/admin  ==> Admin page + login
/calendar ==> Calendar + login


```
